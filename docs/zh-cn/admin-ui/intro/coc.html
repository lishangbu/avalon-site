添加菜单 ​

菜单是基于路由生成的，而路由又是基于unplugin-vue-router生成的，完全是基于文件系统的。 因此，要想了解菜单的生成，需要添加必要的meta原数据 其中，包括顶层菜单与子菜单， 子菜单必须包含:

  • title 渲染菜单标题
  • icon 渲染菜单图标 顶层菜单必须包含: 顶层菜单和子菜单的区别是
  • 子菜单是pages下的一个个模块，用文件夹区分
  • 顶层菜单是pages下对应模块的同名vue文件 具体解析规则参考官方文档

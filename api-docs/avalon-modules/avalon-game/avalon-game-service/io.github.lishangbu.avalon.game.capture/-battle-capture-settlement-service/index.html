BattleCaptureSettlementService

interface BattleCaptureSettlementService

捕捉资源账本结算服务。

职责边界：

  • 读取 battle engine 在 session 中留下的捕捉资源账本；

  • 完成库存校验与道具扣减；

  • 在最终捕捉成功时落库 owned creature；

  • 产出面向上层编排的捕捉结算结果。

这样 avalon-game-service 只需要关注“什么时候结算”， 不再直接处理捕捉业务细节。

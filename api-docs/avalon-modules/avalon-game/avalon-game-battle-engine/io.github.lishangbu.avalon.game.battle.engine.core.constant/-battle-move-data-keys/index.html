BattleMoveDataKeys

object BattleMoveDataKeys

effect data 中 battle 主流程会直接读取的标准字段键集合。

设计意图：

  • 把 "type"、"damageClass"、"target" 这类高频字段键从执行链中抽离；

  • 保持 move resolution、target resolver、critical hit 判定读取同一组字段名；

  • 为后续接 CSV/JSON 装配层时提供一处统一的语义落点。
